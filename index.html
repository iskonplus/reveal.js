Pseudo-classes.

A CSS pseudo-class is a keyword added to a selector that specifies a special state of the selected element(s)

For example:

what color the active link will have or when hovering over it.

How does a pseudo-class differ from a regular class?

  • We don't need to invent the names of pseudo-classes ourselves, we just need to know them.
  • The name of the pseudo-class is added with a colon to the selector, while the name of the regular class is specified inside the HTML markup tag.

Pseudo-classes can be divided into several groups

  • Pseudo-classes for links.
  • Pseudo-classes for input.
  • Pseudo-class for filter and sample.

Pseudo-classes for links:

  • :hover - when hovering the cursor.
  • :link - link not visited.
  • :active - state when the link is clicked.
  • :visited - visited link.

The idea is that each link state has its own CSS styles.

For example, we can change the link color to red when the cursor hovers over it.


							a:hover{
								color: red;
							}

Pseudo-classes for input:

  • :focus - changes the style for the field that received focus.
  • :checked - applies to the checkbox and radio elements of the form when they are in the enabled state.
  • :empty - represents an empty element.
  • :invalid - if not valid input.
  • :valid - if valid input.

And now examples...

:focus

By default, the browser makes an outline inside the input that falls into focus. We cancel the default stroke and set our own styles.


							input:focus {
								outline: none;
								border: 2px solid green;
							}

:empty

Represents an empty element, such as " span".


							span:empty {
								background: blue;
								width: 30px;
								height: 30px;
								display: block;
							}

:valid

If validation was successful, we will switch the field color to green.


							input:valid {

								background: green;
							
							}

Pseudo-classes for filter and sample:

  • :first-child - sets CSS styles for the first child element..
  • :last-child - sets CSS styles for the last child element.
  • :nth-child - accessing an element using an ordinal number.

And again examples...

:first-child

Add a gray background to the first item in the list.


							<ul>
								<li>child 1</li>
								<li>child 2</li>
								<li>child 3</li>
								<li>child 4</li>
								<li>child 5</li>
							</ul>

							li:first-child {
								background: grey;
							}

:nth-child

Let's take the same list as an example. What should I do if I need to select the second item in the list instead of the first or last one?



							// the second element is white
							li:nth-child(2) {
								color: #fff;
							 }
							
							// the third element is black
							 li:nth-child(3) {
								 color: #000;
							 }

							 // even elements are gray
							 li:nth-child(even) {
								 color: #ccc;
							 }

Pseudo-elements.

A CSS pseudo-element is a keyword added to a selector that lets you style a specific part of the selected element(s).

Use the Space key to navigate through all slides.

For example:

can be used to change the font of the first line of a paragraph.

It is important to remember!

You can use only one pseudo-element in a selector. It must appear after the simple selectors in the statement.


							selector::pseudo-element {
								property: value;
							  }

Note

As a rule, double colons (::) should be used instead of a single colon (:). This distinguishes pseudo-classes from pseudo-elements. However, since this distinction was not present in older versions of the W3C spec, most browsers support both syntaxes for the original pseudo-elements.

pseudo elements can be divided into several groups

  • Pseudo-elements for text effects.
  • Pseudo-elements for generated content.
  • Pseudo-elements for tools for highlighting and highlighting.

Pseudo-elements for text effects:

  • ::first-letter - applies styles to the first letter of the first.
  • ::first-line - applies styles to the first line of a block-level element.

For example, make the first letter of each paragraph red and large.


							p::first-letter {
								color: red;
								font-size: 130%;
							  }

Pseudo-elements for generated content:

  • ::before - creates a pseudo-element that is the first child of the selected element.
  • ::after - creates a pseudo-element that is the last child of the selected element.
  • ::placeholder - represents the placeholder text in an <input> or <textarea> element.
  • ::marker - selects the list item marker field.
